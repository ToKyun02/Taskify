## 🖐️ 프로젝트 소개

**Taskify**는 일정 관리 기능을 지원하는 웹 애플리케이션입니다. 

초대 기능을 활용하여 다른 사용자와 일정 관리를 공유할 수 있습니다.

다른 사람들과 일정을 공유하여 성공적으로 **Task**를 해결할 수 있습니다.
